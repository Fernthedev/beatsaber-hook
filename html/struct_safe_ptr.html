Detailed Description

template<class T>
struct SafePtr< T >

Represents a C++ type that wraps a C# pointer that will be valid for the entire lifetime of this instance. This instance must be created at a time such that il2cpp_functions::Init is valid, or else it will throw a CreatedTooEarlyException.

Template Parameters
TThe type of the instance to wrap.

Copy constructor copies the HANDLE, that is, the held pointer remains the same. Note that this means if you modify one SafePtr's held instance, all others that point to the same location will also reflect this change. In order to avoid a (small) performance overhead, consider using a reference type instead of a value type, or the move constructor instead.
