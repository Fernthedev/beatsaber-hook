void* gc_alloc_specific ( size_t  sz)

Returns an allocated instance of the provided size that will not be written over by future GC allocations and holds references. You MUST use the gc_free_specific function defined here to destroy it. This function fallsback to calloc if no GC_Alloc or GC_Free implementations are found via xref/sigscan.
